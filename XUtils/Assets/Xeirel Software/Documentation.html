XUtils ñ Smart Unity Utilities

XUtils is a collection of lightweight, code-first utility helpers for Unity. It focuses on making common tasks (UI, strings, formatting, input, etc.) easier, while staying non-intrusive and dependency-free.

Target: Unity C# scripts
Style: Drop-in utilities, no custom editor windows required.

1. Installation

  • Import the XUtils folder into your Unity project.
  • Make sure TextMeshPro is installed (for TMP-related utilities).
  • Open the sample scene(s) under:
    Assets/Xeirel Software/XUtils/Demo/

2. Overview of Features

  • UI Utilities
    • Hover detection and hover timing for UI elements.
    • Double-click handling for UI elements.
    • Simple conditional inspector fields (editor only).
  • String & Text Utilities
    • Colorized JSON output for TMP.
    • Smart number/amount formatting.
    • Money highlighting with color.
    • Rich Text helpers (bold, italic, color, size).
    • Hex/Base64 encode & decode for structs.
    • TMP tag stripping, whitespace removal.
  • Math & Geometry Utilities
    • Index conversion helpers for grids (1D <-> 2D/3D/4D).
    • Smoothed lerp helpers with snapping.
    • Vector helpers and bounds/world-space utilities.
    • Random float and bounds sampling.
  • Collections & Utility Extensions
    • Safe accessors and search helpers for IEnumerable/List/Dictionary.
    • Chunking, DistinctBy, MinBy/MaxBy equivalents.
    • LayerMask helpers and simple index wrapping.
  • Resources & Asset Helpers
    • Cached wrapper for Resources.Load / Resources.LoadAll.
  • Debug / Developer Helpers
    • Stack trace parsing to extract the calling code context.
    • String morphing utilities to create text transition effects.

3. UI Utilities

3.1 XUIHoverUtility

Namespace: XUtils.UIUtils
Static class: XUIHoverUtility

Centralized hover detection for Unity UI (EventSystem-based). It tracks the currently hovered GameObject and how long the pointer has been hovering over it.

Key members:

  • GameObject XUIHoverUtility.CurrentHovered
  • float XUIHoverUtility.SecondsSinceHoverEnter
  • bool XUIHoverUtility.IsHovered(GameObject go)
  • bool XUIHoverUtility.IsHovered(RectTransform rt)
  • bool XUIHoverUtility.IsHoveredIncludingSelfAndChildren(RectTransform target)
  • Events:
    • Action<GameObject> XUIHoverUtility.OnHoverEnter
    • Action<GameObject> XUIHoverUtility.OnHoverExit
    • Action<GameObject> XUIHoverUtility.OnHoverChanged

Usage:

  • Call XUIHoverUtility.Update() once per frame from a central MonoBehaviour (for example, in your UI managerís Update method).
  • Read CurrentHovered and SecondsSinceHoverEnter anywhere in your code.

3.2 UIDoubleClickHandler

Namespace: XUtils
File: UIDoubleClickHandler.cs

Provides simple, reusable double-click detection for UI elements.

Core concepts:

  • IPointerDoubleClickHandler ñ interface used for double-click callbacks.
  • UIDoubleClickHandler ñ a MonoBehaviour you can add to UI elements to receive double-clicks via an inspector event (UnityEvent<PointerEventData>) and/or an overridable method (HandleDoubleClick).

Usage ñ Inspector Only:

  • Add UIDoubleClickHandler to a UI GameObject (for example, a button).
  • In the inspector, hook up the OnDoubleClick event to any method you like.

Usage ñ Inheritance (code):

  • Derive from UIDoubleClickHandler and override HandleDoubleClick to implement custom behavior.

3.3 ConditionalField (Editor Only)

Namespace: XUtils.EditorUtils
Files: ConditionalFieldAttribute.cs, TestConditionalField.cs (demo)

Provides a simple attribute and property drawer to show/hide fields in the inspector based on other field values, without writing a custom editor.

Examples:

// Toggle-based:
public bool ShowAdvancedSettings;
[ConditionalField(nameof(ShowAdvancedSettings))]
public bool ShowFPS;

// Enum-based:
public TestEnum MyTestEnum;
[ConditionalField(nameof(MyTestEnum), typeof(TestEnum), TestEnum.OptionB)]
public string AdvancedOptionForB;

This keeps inspectors clean while still being easy to configure from code.

4. String & Text Utilities

Namespace: XUtils.StringUtils
Static class: XString

4.1 JSON Colorization (for TextMeshPro)

  • ColorizeJson(string json) returns a TMP-compatible string with <color> tags applied to:
    • Keys
    • Strings
    • Numbers
    • Booleans
    • null
    • Symbols ({}, [], ,)

4.2 Smart Number Formatting

  • ToSmartString<T>(T number, int maxDecimals = 2, bool useThousandSeparator = false)
    • Uses current culture decimal / thousand separators.
    • Trims unnecessary decimals if the value is an integer.
    • Optionally uses thousand separators.

4.3 Money Highlighting

  • GiveMoneyColor(this string input)
    • Scans the string for amounts followed by a currency symbol (for example, $) and wraps them in <color> tags to highlight them.

4.4 Rich Text Helpers

  • Color(this string text, string color) / Color(this string text, Color color)
  • Bold(this string text)
  • Italic(this string text)
  • Size(this string text, string size)

These helpers simplify generating TMP rich text at runtime.

4.5 String Morphing

  • MorphString(string a, string b, float t)
  • MorphStringRandom(string a, string b, float t)

These methods help create text transition effects between two string values, controlled by t (0ñ1).

4.6 Misc String Helpers

  • RemoveWhitespaces()
  • RemoveTMPTags()
  • TrimStartOnce()
  • ConvertToColor()
  • SetTmpText()

5. Math & Geometry Utilities

Namespace: XUtils.MathUtils
Static class: XMath

5.1 Index Helpers for Grids

  • To2D/To3D/To4D(this int index, int2/int3/int4 size)
  • To1D(this int2/int3/int4 index, int2/int3/int4 size)

These helpers convert between 1D indices and multi-dimensional indices, which is very useful for grid-based games and data structures (tilemaps, voxel grids, etc.).

5.2 Lerp Helpers with Snapping

  • Lerp(float current, float target, float t, float snapDistance = 0.0001f)
  • Lerp(Vector2/Vector3/Quaternion current, target, float t, float snapDistance = 0.0001f)
  • Extension variants like LerpTo(ref float val, float to, float time, int? r = null)

They provide convenient smoothing with optional snapping when close to target, reducing small floating point drift.

5.3 Numeric Helpers

  • Clamp/Min/Max/Abs/Pow/Distance for float/int/double/long
  • Remap(this float/double valueIn, baseMin, baseMax, limitMin, limitMax)
  • NotNaN() helpers to guard against NaN values.

These are small quality-of-life wrappers around common math operations tailored to Unity projects.

5.4 Vector Helpers

  • ToInt()/ToFloat() conversions for Vector2/3 and Vector2Int/3Int
  • Multiply (component-wise scale)
  • Distance/DistanceEps/DistanceR
  • Dot/Angle
  • R/Abs/Pow for Vector2/Vector3
  • Swizzle-like helpers: XY(), YZ(), XZ(), XYZ(), etc.
  • Bounds transform helpers:
    • Calc(this Matrix4x4, Vector3)
    • TransformBoundsToWorld(this Bounds, Transform)
    • TransformMatrix(this Bounds, Matrix4x4)

These focus on typical gameplay and geometry tasks like snapping, distance checks with tolerance, and transforming local bounds into world-space AABBs.

5.5 Random Helpers

  • System.Random.NextFloat(float minValue, float maxValue)
  • Bounds.GetRandomPosition()

Useful for sampling random positions inside bounds or generating random floats via System.Random.

6. Struct Serialization Helpers

  • GetHex<T>(this T value) / ResolveHex<T>(this string hex)
  • EncodeToBase64<T>(T value) / DecodeFromBase64<T>(string base64)

These helpers are intended for advanced scenarios where you need to inspect or serialize value types at the byte level.

7. Collections & Utility Extensions

Namespace: XUtils.CollectionsUtils
Static class: XCollections

Highlights:

  • Safe access and search:
    • GetAt(index) ñ safe element access by index.
    • Find(predicate) / TryFind(List<T>, Predicate<T>, out T)
    • FindWithIndex(predicate)
    • FindIndex(predicate)
    • GetKeyFromValue on IDictionary.
  • Sequence utilities:
    • Enumerate(Func<int, T> selector, int count)
    • ConvertTo (projection via Converter<TSource, TOut>)
    • SelectEach (apply selector to each item)
    • ToSeparatedString (simple join helper)
  • Random and chunking:
    • RandomElement (optional Random)
    • Chunk(size) ñ split sequence into fixed-size arrays.
  • Distinct and min/max by key:
    • DistinctBy(keySelector)
    • MaxBy/MinBy(keySelector)
  • Counting and index helpers:
    • TryGetNonEnumeratedCount
    • Index (enumerate with indices)
    • IndexOf(predicate)
    • WrapIndex ñ safe index wrap around list/array length.
  • LayerMask helper:
    • Contains(this LayerMask, int layer)

These helpers give you many LINQ-like features in a Unity-friendly, extension-based style and are especially handy when working with collections in gameplay and tools code.

8. Resources & Asset Helpers

Namespace: XUtils.UnityUtils
Static class: XResources

Provides a thin caching layer around Unityís Resources API:

  • Load<T>(string path)
  • TryLoad<T>(string path, out T asset)
  • LoadFirst<T>(string path)
  • TryLoadFirst<T>(string path, out T asset)
  • LoadAtPath<T>(string folder, string name)
  • TryLoadAtPath<T>(string folder, string name, out T asset)
  • LoadAll<T>(string path)
  • Unload(string path)
  • UnloadAll()
  • Clear() ñ clear cache references.

The cache avoids repeated Resources.Load calls and offers debug logging (via XUtilsBase) when enabled.

9. Debug & Stack Trace Helpers

  • ExtractCallerFromStack(this string stackTrace)

Parses a stack trace string and tries to extract the most relevant line from project code (for example, under Assets/...). This is useful in custom log or diagnostic UIs.
